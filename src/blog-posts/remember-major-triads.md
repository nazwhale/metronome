Use these 7 words to instantly recall any major [triad](/dictionary/triad) (Root–3rd–5th):

| Root | Word | Triad | Meaning |
|------|------|-------|---------|
| A | **ACCE**nt | A–C#–E | emphasis |
| B | **B**i**DD**e**F**ord**s** | B–D#–F# | city in Maine/town in Devon |
| C | **CE**ilin**G** | C–E–G | ceiling |
| D | **D**u**FF**m**A**n | D–F#–A | Simpsons character |
| E | **EGGB**ox | E–G#–B | where eggs live |
| F | **FAC**e | F–A–C | face |
| G | **G**ar**B**e**D** | G–B–D | dressed |

### The Rules

1. **Letters give Root → 3rd → 5th** (in order)
2. **Double letter = sharp** (CC = C#, DD = D#, FF = F#, GG = G#)
3. **Only B:** final S = F# (sharp 5th)
4. **Bonus:** 3rd → Root = 6th, 5th → Root = 4th

That's it. Seven words, all twelve notes covered.
